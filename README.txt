Tal como tínhamos falado venho por este meio pedir que executes o jogo da forca,  utilizando as seguintes frameworks:

·         Servlets + Arrayes (dados);

·         Servlets + ligação a BD

·         Servlets + Hibernate;

·         Struts + Hibernate;

·         Spring + Hibernate;



No final de implementares o jogo da forca em cada uma das frameworks faz uma breve apresentação dessa framework.



A base de dados deverá ter 4 tabelas, a saber:

·         Palavra;

·         Categoria;

·         Categoria/Palavra;

·         Dificuldade;



O utilizador antes do jogo começar deverá escolher a dificuldade e a categoria, caso o utilizador não escolha nenhuma das opções, deverá ser feito um random as categorias e à dificuldade.
